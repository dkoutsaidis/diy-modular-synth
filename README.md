# diy-modular-synth
This repo hosts schematics and panel designs for varius DIY [eurorack](https://en.wikipedia.org/wiki/Eurorack) modules.

![synth](https://user-images.githubusercontent.com/24361657/112227922-10ee2880-8c39-11eb-97c4-7faf721770ca.png)

### Repo Structure
- The main branch hosts already built and tested modules.
- The develop branch hosts modules that are currently being developed and/or tested.

### Software
- Schematics and PCBs are created with the open-source [KiCad](https://kicad.org/) software (default libraries).
Specifications for the Power Supply, System Bus, CV & MIDI signals can be found [here](http://www.doepfer.de/a100_man/a100t_e.htm).
- Panel designs are created with the open-source [LibreCAD](https://librecad.org/) software.
Custom components can be found inside the [Panels Design Library](/Panels%20Design%20Library) folder. Specifications for eurorack panels can be found [here](http://www.doepfer.de/a100_man/a100m_e.htm).
